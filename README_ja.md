**Twitter動画ランキングサイトの煩わしい多段リダイレクト、画面中に飛び交うポップアップ広告、そして極端に遅い動画バッファリングにイライラしていませんか？**

X-Flowは、`truvaze.com`、`pektino.com`、`x-ero-anime.com`などのミラーサイト向けに設計された**ユーザースクリプト（Userscript / Tampermonkey）**です。

ページを開いた瞬間に強力なサンドボックスで乗っ取り、**広告、ポップアップ、悪質なリダイレクトを物理的に完全ブロックします。動画のみを直接抽出し、純粋で極めてスムーズなTikTok風フルスクリーン縦型プレイヤーを提供します**。

余計なページ遷移も広告もありません。最高に純粋な動画ブラウジング体験をお届けします！

---

## 🌟 主な機能と特徴 (Why X-Flow?)

*   **🚫 広告と悪質リダイレクトの完全排除**：ポップアップ広告や強制リダイレクト、トラッキングスクリプトをブラウザレベルで強制ブロックし、クリーンで安全な閲覧環境を提供します。
*   **🎬 TikTok風の超高速動画ブラウジング**：カードをクリックするだけでフルスクリーン縦型プレイヤーが起動。マウスホイール、キーボード矢印キー（↑/↓）、スマートフォンでの上下スワイプで流れるように動画を切り替えられます。
*   **💾 再生位置の自動記憶（レジューム再生）**：各動画の再生位置（秒単位）を自動保存。再開時は瞬時にその位置から再生され、ロード中の進捗バーの点滅を防ぐビジュアル配慮も施されています。
*   **⚡ 0秒のシームレス切り替え（戻り再生）**：スマートなバックグラウンドキャッシュにより、隣接する動画間を戻って再再生する際、通信バッファを待つことなく**完全に瞬時（0秒）に再生開始**します。
*   **🚀 3倍速バッファリング**：通信速度を制限する不要なプロキシ層を無効化。バックグラウンドでのTCP/SSL事前接続技術と組み合わせることで、動画のバッファリング速度を300%向上させました。
*   **📂 投稿者パネルと一括ダウンロード**：左スワイプで投稿者のプロフィールと動画リストを表示。動画を選択して、透かしなしの動画直リンク（CDN URL）を一括コピーし、外部ダウンロードツールへ簡単にインポートできます。
*   **📱 モバイル端末ジェスチャーの最適化**：iOS Safariなどで発生する「1回目のタップで要素が縮小（ホバー判定）され、2回目でようやく再生される」というWebKit特有のバグを解消。タップ遅延なく一発起動します。

## 🎮 操作方法 (Controls)

| 操作方法 | 機能 |
| :--- | :--- |
| **🖱️ マウスホイール / ⌨️ 矢印キー ↑/↓** | 前後の動画に切り替え |
| **⌨️ スペース / Enter / 🖱️ 動画をクリック** | 再生 / 一時停止 |
| **👆 スマートフォン上下スワイプ** | 前後の動画に切り替え |
| **👆 スマートフォン左スワイプ** | 投稿者パネル＆ダウンロードリンクの表示 |
| **👆 スワイプ右 / ⌨️ Esc / ❌ ボタン** | プレイヤーを閉じる |
| **⚡ 再生速度の調整** | 0.5倍から2.0倍までの多段階倍速調整 |
| **🔘 未読フィルター** | 視聴済みの動画を非表示にし、新規動画のみを表示 |

## 🛠️ インストール方法 (Installation)

### ステップ 1：拡張機能のインストール
お使いのブラウザに拡張機能 **Tampermonkey** をインストールします：[https://www.tampermonkey.net/](https://www.tampermonkey.net/)

### ステップ 2：スクリプトのインストール
以下のいずれかの配布サイトからインストールしてください：
*   **Greasy Fork**：[https://sleazyfork.org/zh-CN/scripts/562631](https://sleazyfork.org/zh-CN/scripts/562631)
*   **国内代替ミラー (OpenUserJS)**：[https://openuserjs.org/scripts/loadingi](https://openuserjs.org/scripts/loadingi)

インストール後、サポートされている以下のURLにアクセスすると、自動的にスクリプトが作動します：
*   `truvaze.com`
*   `pektino.com`
*   `x-ero-anime.com`

## 💻 ローカル開発とビルド (Development)

ソースコードの修正やご自身でのコンパイルは、以下の手順で行えます：

1. **リポジトリをクローン / ダウンロード**
2. **依存関係のインストール**
   ```bash
   npm install
   ```
3. **ローカル開発サーバーの起動 (HMRホットリロード対応)**
   ```bash
   npm run dev
   ```
   起動後、ターミナルに表示される `http://localhost:5173/` などのアドレスから `.user.js` をTampermonkeyに読み込ませることで、コード修正がリアルタイムにブラウザへ反映されます。
4. **本番用ビルドの作成**
   ```bash
   npm run build
   ```
   ビルド完了後、配布用のスクリプトが `dist/userscript/x-flow.user.js` に出力されます。
