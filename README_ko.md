**트위터 랭킹 사이트의 복잡한 다단계 리디렉션, 화면을 가득 채우는 팝업 광고, 그리고 극도로 느린 동영상 버퍼링에 아직도 고통받고 계십니까?**

X-Flow는 `truvaze.com`, `pektino.com`, `x-ero-anime.com` 등의 미러 사이트에 최적화된 **유저스크립트(Userscript / Tampermonkey)**입니다.

페이지가 열리는 순간 강력한 샌드박스로 페이지를 제어하여 **광고, 팝업 및 악성 리디렉션을 완전히 차단하고, 동영상 원본을 직접 추출하여 깔끔하고 몰입감 넘치는 TikTok/쇼츠 스타일의 풀스크린 세로형 플레이어로 리패키징합니다**.

불필요한 페이지 이동도, 광고도 없는 가장 순수한 동영상 스와이프 감상 환경을 경험해 보세요!

---

## 🌟 주요 기능 및 특징 (Why X-Flow?)

*   **🚫 모든 광고 및 리디렉션 차단**: 팝업 광고, 백그라운드 리디렉션, 추적 스크립트를 브라우저 수준에서 강력하게 차단하여 100% 안전하고 쾌적한 환경을 제공합니다.
*   **🎬 TikTok 스타일 동영상 탐색**: 카드를 클릭하면 즉시 풀스크린 세로형 플레이어가 실행됩니다. 마우스 휠, 키보드 방향키(↑/↓) 또는 스마트폰 스와이프를 사용하여 동영상을 물 흐르듯 전환할 수 있습니다.
*   **💾 재생 위치 자동 기억 (이어보기)**: 각 동영상의 시청 기록을 초 단위로 자동 저장합니다. 다시 해당 동영상으로 돌아오면 버퍼링 대기 시간 동안 진행 표시줄과 시각 정보를 미리 렌더링하여 매끄럽게 이어보기를 지원합니다.
*   **⚡ 버퍼링 없는 0초 전환 (뒤로가기 재생)**: 스마트 백그라운드 캐시 기술을 통해 이전 동영상으로 다시 스와이프할 때, 재버퍼링 대기 없이 **0초 만에 즉시 재생**됩니다.
*   **🚀 3배 빠른 버퍼링 속도**: 동영상 다운로드 속도를 제한하는 호스트 사이트의 백그라운드 프록시 네트워크 레이어를 무력화하고, 백그라운드 TCP/SSL 사전 연결을 결합하여 동영상 버퍼링 속도를 300% 가속합니다.
*   **📂 크리에이터 패널 및 일괄 다운로드**: 왼쪽으로 스와이프하여 해당 크리에이터의 프로필과 동영상 목록을 불러올 수 있습니다. 동영상을 일괄 선택하여 워터마크 없는 CDN 직링크 URL을 복사한 뒤 다운로드 매니저에 쉽게 붙여넣을 수 있습니다.
*   **📱 모바일 제스처 최적화**: iOS Safari 등 모바일 WebKit 엔진에서 "첫 번째 터치 시 카드만 축소(Hover)되고, 두 번째 터치해야 재생되는" 고질적인 버그를 완전 해결하여 한 번의 터치로 지연 없이 즉시 실행됩니다.

## 🎮 조작 방법 (Controls)

| 조작 방식 | 기능 |
| :--- | :--- |
| **🖱️ 마우스 휠 / ⌨️ 방향키 ↑/↓** | 이전 / 다음 동영상 전환 |
| **⌨️ 스페이스바 / Enter / 🖱️ 동영상 클릭** | 재생 / 일시정지 |
| **👆 스마트폰 상하 스와이프** | 다음 / 이전 동영상 전환 |
| **👆 스마트폰 좌측 스와이프** | 크리에이터 패널 열기 및 다운로드 링크 추출 |
| **👆 스마트폰 우측 스와이프 / ⌨️ Esc / ❌ 버튼** | 플레이어 닫기 |
| **⚡ 배속 조절** | 0.5배속부터 2.0배속까지 다양한 배속 조절 지원 |
| **🔘 시청 완료 필터 토글** | 이미 시청한 동영상을 목록에서 제외하고 새 동영상만 보기 |

## 🛠️ 설치 방법 (Installation)

### 1단계: 브라우저 확장 기능 설치
브라우저에 **Tampermonkey** 확장 프로그램을 설치합니다: [https://www.tampermonkey.net/](https://www.tampermonkey.net/)

### 2단계: X-Flow 스크립트 설치
아래 배포 사이트 중 하나를 선택하여 설치해 주세요:
*   **Greasy Fork (성인용/SleazyFork)**: [https://sleazyfork.org/zh-CN/scripts/562631](https://sleazyfork.org/zh-CN/scripts/562631)
*   **국내 접속용 백업 링크 (OpenUserJS)**: [https://openuserjs.org/scripts/loadingi](https://openuserjs.org/scripts/loadingi)

설치 완료 후, 지원되는 아래 도메인에 접속하면 스크립트가 자동으로 작동합니다:
*   `truvaze.com`
*   `pektino.com`
*   `x-ero-anime.com`

## 💻 로컬 개발 및 빌드 (Development)

프로젝트 코드를 직접 수정하거나 빌드하려면 다음 단계를 따르세요:

1. **프로젝트 클론 및 다운로드**
2. **패키지 설치**
   ```bash
   npm install
   ```
3. **로컬 개발 서버 실행 (HMR 자동 반영 지원)**
   ```bash
   npm run dev
   ```
   실행 후 터미널에 표시되는 `http://localhost:5173/` 형태의 주소를 Tampermonkey에 유저스크립트로 추가하면, 소스 코드를 수정할 때마다 웹 브라우저에 실시간으로 반영됩니다.
4. **배포용 빌드 컴파일**
   ```bash
   npm run build
   ```
   빌드가 완료되면 배포 가능한 스크립트 파일이 `dist/userscript/x-flow.user.js` 경로에 생성됩니다.
